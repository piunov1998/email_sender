Установка:
    1. Скачать и установить Python3 с официального сайта https://www.python.org/downloads/
    2. Открыть консоль (cmd.exe) и выполнить команду: pip install -U -r requirements.txt

Первоначальная настройка:
    1. Запусить файл start.py
    2. Будет создана папка userdata, а также файла для ваших данных.
    3. Необходимо заполнить по образцу файл logins.json вашими email'ами с которых будет осущесвтляться рассылка.
    4. Файл message.txt содержит в первой строке заголовок, в последующих - тело электронного письма.
    5. (не обязательно) Открыть через любой текстовый редактор файл reform.py и отредактировать пути файлов с старой таблицей и новой. Выполнить скрипт.
    6. Поместить базу email-адресов в папку userdata с названием data.xlsx

Настройка параметров отправки:
    1. Файл settings.json

Использование:
    1. После выполнения первоначальной настройки запустить файл start.py.
