Установка:
    1. Скачать и установить Python3 с официального сайта https://www.python.org/downloads/
    2. Поместить папку с программой в удобном месте, можно создать ярлык файла start.bat для более удобного запуска.
    3. Дальнейшая установка произойдет при первом запуске.

Первоначальная настройка:
    1. Запусить файл start.py
    2. Ввести запрашиваемые параметры(в будущем их можно будет поменять в файле settings.json)
    3. Будет создана папка userdata, а также файла для ваших данных.
    4. Необходимо заполнить по образцу файл logins.json вашими email'ами с которых будет осущесвтляться рассылка.

Настройка параметров отправки:
    1. Необходимо открыть settings.json в любом текстовом редакторе и изменить необходимые параметры(можно удалить для сброса настроек).

Использование:
    1. Тема и текст сообщения редактируются в файле message.txt в папке userdata 
        (в первой строке заголовок, в последующих - тело электронного письма).
    2. Файлы для закрепления в письма необходимо поместить в папку ./userdata/attachments
    3. После выполнения первоначальной настройки запустить файл start.bat.
